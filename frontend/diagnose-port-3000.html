🔍 Port 3000 OAuth Callback 진단

문제: 팝업에서 로그인 완료해도 OAuth 데이터가 sessionStorage에 저장되지 않음

원인: Port 3000의 MAX Platform 콜백 페이지가 예상과 다른 방식으로 작동

🎯 진단 목표

  • Port 3000 콜백 페이지가 어떤 storage keys를 사용하는지 확인
  • 실제로 어떤 postMessage나 communication 방식을 사용하는지 파악
  • BroadcastChannel, localStorage, 또는 다른 방식 사용 여부 확인
  • Cross-origin recovery 메커니즘 개선 방향 결정
