🕵️ OAuth Redirect Investigation

목적: OAuth 서버가 실제로 어디로 리다이렉트하는지 확인하고, 그 페이지에서 토큰을 찾을 수 있는지 조사

방법: 실제 OAuth 플로우를 시작하고 팝업의 모든 URL 변화를 추적

🎯 조사 계획

  1. OAuth 팝업 열기 (실제 "다른 사용자로 로그인" 플로우)
  2. 팝업의 URL 변화를 주기적으로 기록
  3. 각 URL에서 사용 가능한 데이터 확인
  4. 최종 리다이렉트 URL과 포트 확인
  5. 해당 페이지의 storage와 DOM 분석
